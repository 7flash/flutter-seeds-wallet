## Getting Started

```
git clone https://github.com/7flash/flutter_seeds_wallet.git
cd flutter_seeds_wallet
flutter pub get
flutter run
```

## Dev configuration

Use the projects git hooks
```
git config core.hooksPath .githooks/
```
